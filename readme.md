# Phaser capacitor boilerplate

This is a starter code to create html games with phaserjs an ionic + capacitor to deploy to iOS and Android platforms.

## Pre-requisites 

- Install node 16+
- Install ionic
``` npm i -g @ionic/cli ```

### Run in web 
``` ionic serve ```

### Run iOS
``` ionic capacitor run ios ```


### Run android
``` ionic capacitor run android ```
